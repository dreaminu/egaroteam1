# 지역별 실거래 인기 평수 찾기

국토교통부 실거래가 공개시스템에서 데이터를 자동으로 가져와서, 사용자가 선택한 시도/연도에서 가장 많이 거래된 **동네 + 평수대**를 찾아주는 Streamlit 웹앱입니다.

이제 수강생은 엑셀 파일을 직접 다운로드하거나 업로드하지 않아도 됩니다.

## 핵심 사용 예

```text
시도: 인천광역시
물건 종류: 연립/다세대
조회 연도: 2025년

[국토부에서 가져와 분석하기]
```

결과:

```text
인천광역시 2025년 기준으로 가장 거래량이 많은 조합은
'OO동 · 12~14평' 입니다.

거래건수
평균 거래금액
평균 평당가
거래량 많은 동네 + 평수대 TOP 20
```

## 수강생 사용 흐름

```text
1. 웹앱 링크 접속
2. 시도 선택: 예) 인천광역시, 서울특별시, 경기도
3. 물건 종류 선택: 예) 연립/다세대
4. 조회 연도 선택: 예) 2025년
5. [국토부에서 가져와 분석하기] 클릭
6. 결과 확인
7. HTML 리포트 또는 엑셀 결과 다운로드
```

## 지원 물건 종류

- 아파트
- 연립/다세대
- 단독/다가구
- 오피스텔
- 분양/입주권
- 상업/업무용
- 토지
- 공장/창고 등

## 이 앱이 내부에서 하는 일

```text
1. 국토교통부 실거래가 공개시스템 접속
2. 국토부 시도 목록 조회 및 선택값의 시도 코드 확인
3. 물건 종류 탭 코드 선택
4. 선택한 연도 1년치 CSV 다운로드 요청
5. 거래 데이터 정리
6. 법정동 + 면적구간별 거래량 집계
7. 가장 거래량이 많은 동네와 평수대 표시
```

공공데이터포털 API 키는 사용하지 않습니다.
국토부 실거래가 공개시스템의 공개 다운로드 흐름을 사용합니다.

## 로컬에서 웹앱 실행

```bash
cd /home/daesung/realestate-analyzer
python3 -m pip install -r requirements.txt
streamlit run app.py
```

브라우저에서 보통 아래 주소로 열립니다.

```text
http://localhost:8501
```

## 수강생에게 링크로 공유하는 방법

가장 쉬운 방법은 Streamlit Community Cloud입니다.

1. GitHub에 이 프로젝트를 올립니다.
2. https://streamlit.io/cloud 에 접속합니다.
3. New app을 누릅니다.
4. GitHub 저장소를 선택합니다.
5. Main file path에 `app.py`를 입력합니다.
6. Deploy를 누릅니다.
7. 생성된 링크를 수강생에게 공유합니다.

공유 링크 예:

```text
https://your-app-name.streamlit.app
```

## 다운로드 결과물

웹앱에서 아래 파일을 받을 수 있습니다.

- HTML 리포트
- 엑셀 결과 파일
- CSV 파일: 고급 다운로드
- JSON 파일: 고급 다운로드

수강생에게는 보통 HTML 리포트와 엑셀 결과만 안내하면 됩니다.

## 중요한 제한사항

- 국토부 사이트 안내 기준으로 시도별 자료제공 계약일자 범위는 최대 1년입니다.
- 국토부 사이트 구조가 바뀌면 자동 수집 기능이 수정되어야 할 수 있습니다.
- 요청이 너무 많으면 사이트 응답이 느려지거나 실패할 수 있습니다.
- 이 도구는 공개 데이터 다운로드 흐름을 이용하며, 공식 통계가 아닙니다.

## 중요한 해석 원칙

이 프로그램은 국토부 실거래가 공개시스템에서 내려받은 데이터만 분석합니다.

- 데이터에 없는 내용은 추측하지 않습니다.
- 재개발 여부는 임의 판단하지 않습니다.
- 전세가율은 매매 데이터만으로 계산하지 않습니다.
- 거래량이 많다고 무조건 투자 추천하지 않습니다.
- “거래가 활발하다”와 “투자 가치가 높다”를 구분합니다.
- 공장/창고, 상업/업무용, 토지는 주거용 아파트·빌라와 가격 해석 기준이 다르므로 별도 현장 확인이 필요합니다.
